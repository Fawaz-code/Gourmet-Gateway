# 🍽️ Golden Fork Steakhouse

  > Premium luxury steakhouse website — Dallas, TX

  ## Tech Stack
  - **React 18** + **Vite** + **TypeScript**
  - **Tailwind CSS v4** + **Framer Motion**
  - **Lucide React** + **react-icons**

  ## Features
  - 🎬 Animated gold shimmer hero headline
  - 🔥 Live specials ticker bar
  - 🤖 TastyAI chatbot assistant
  - 📱 Fully responsive (mobile + desktop)
  - 🖼️ Gallery with lightbox
  - ⭐ Testimonials carousel
  - 📞 WhatsApp integration
  - 📋 Contact form with validation

  ## Sections
  Navbar · Hero · Stats · About · Menu · Gallery · Testimonials · Contact · Footer

  ## Run Locally
  ```bash
  pnpm install
  pnpm --filter @workspace/golden-fork run dev
  ```
